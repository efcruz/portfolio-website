Featured Projects

myFlix (server-side)

Server-side of a web application that gives users access to movie information, such as title, genre, director, etc. The user can sign up, update personal information and add or remove favorite movies. It was built with JavaScript running on Node.js with Express and includes my own RESTful API database in MongoDB, supported by Mongoose for business logic. User authentication was built with Passport. It is deployed in Heroku.

GitHub Repository / Demo

myFlix (client-side)

Single page application (SPA) that handles data through an API also built by me (see myFlix - server side). It gives access to movie information, such as title, genre, director, etc. The user can sign up, update personal information and add or remove favorite movies. It was made with JavaScript on React with Redux for state handling. UI designed with Bootstrap.

GitHub Repository / Demo

Meet App

Progressive web application (PWA) built with React that allows users to search for a city and get a list of events hosted in that city. The events are fetched from Google Calendar API using AWS Lambda as authorisation server for issuing OAuth2 authentication. It was developed following test-driven development (TDD) supported by Jest.
