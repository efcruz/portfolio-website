myFlix API

Server-side for a web application that gives users access to movie information, such as title, genre, director, etc. The user can sign up, update personal information and add or remove favorite movies. It was built with JavaScript running on Node.js with Express and includes my own RESTful API database in MongoDB, supported by Mongoose for business logic. User authentication was built with Passport. It is deployed in Heroku.

Meet App

Progressive web application (PWA) that allows users to search for a city and get a list of events hosted in that city. The events are fetched using Google Calendar API using AWS Lambda as authorisation server for issuing OAuth2 authentication. It was built on React through test-driven development (TDD) handled by Jest.

GitHub Repository

Chat App
